# BODY-MAKE CHIPS 発売前クリック集計

このフォルダは、LPの「試してみたいに1票入れる」クリック数と、任意入力のメールアドレスを Google スプレッドシートに保存するためのものです。

## 1. Googleスプレッドシートを作る

新しいGoogleスプレッドシートを作成します。名前は例として `BODY-MAKE CHIPS 事前反応集計` などでOKです。

## 2. Apps Scriptを貼る

スプレッドシート上部メニューから `拡張機能` → `Apps Script` を開きます。

エディタ内のコードをすべて消して、`tracking/google-apps-script.js` の中身を貼り付けます。

## 3. 初期設定を実行する

Apps Script画面上部の関数選択で `setup` を選び、実行します。

初回はGoogleの権限確認が出ます。許可すると、スプレッドシートに以下の2シートが作られます。

- `launch_events`: クリックやメール入力の生データ
- `summary`: クリック数・メール数の集計

## 4. Webアプリとして公開する

Apps Script右上の `デプロイ` → `新しいデプロイ` を選びます。

- 種類: `ウェブアプリ`
- 実行ユーザー: `自分`
- アクセスできるユーザー: `全員`

デプロイ後に表示される `ウェブアプリURL` をコピーします。

## 5. LP側にURLを入れる

`script.js` のこの行に、コピーしたURLを入れます。

```js
const launchInterestEndpoint = "ここにウェブアプリURL";
```

これで公開後、クリックとメール入力がGoogleスプレッドシートに記録されます。

## 確認方法

スプレッドシートの `summary` を見ると、以下が確認できます。

- 試してみたいクリック数
- 通知希望メール数
- ユニーク通知希望メール数

生データを見たい場合は `launch_events` を確認してください。
