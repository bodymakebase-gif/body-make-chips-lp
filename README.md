# BODY-MAKE CHIPS LP

BODY-MAKE CHIPSの公式ランディングページです。

- 公開基盤：GitHub Pages
- 正式URL：https://body-make-chips.jp/
- 正式ドメイン：`body-make-chips.jp`（wwwなし）
- 公開元：`main`ブランチのリポジトリルート

## 更新方法

サイト改修は従来どおり、このGitHubリポジトリ内のHTML、CSS、JavaScript、画像を編集します。公開元ブランチへ反映すると、GitHub Pagesのビルド後に独自ドメイン側へも反映されます。

通常のコンテンツ修正ではDNS設定を変更しません。リポジトリルートの`CNAME`ファイルは削除したり、別のドメインへ書き換えたりしないでください。
